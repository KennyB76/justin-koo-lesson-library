1. The Three Rules — Restated

Move the entire stack of rings from one tower to another, where:
  1. Rule I: One move = one ring, one tower → another tower.
  2. Rule II: Never place a larger ring on a smaller one.
  3. Rule III: Goal — move the entire stack to a different tower (typically far-left → far-right).
Key observation: Rule II forces a recursive structure. To move the bottom ring, every ring above it has to be parked somewhere first.

2. Problem 1 — Minimum Moves for 8 Rings

Determine the minimum number of moves required to accomplish this "stack exchange" given 8 rings.

2.1 Build the pattern from small cases

Let $T(n)$ = minimum number of moves needed to relocate $n$ rings from one tower to another.

CASE n=1 One ring. Just move it: A → C. $T(1) = 1$.
CASE n=2 Small ring on top, big ring on bottom, both on A.
  1. Small ring: A → B (out of the way)
  2. Big ring: A → C
  3. Small ring: B → C (drops on top of the big ring)
$T(2) = 3$.
CASE n=3 Three rings on A. To move the bottom ring (ring 3) from A to C, the top two must be parked on B first.
  1. Move top 2 rings from A → B: that's $T(2) = 3$ moves.
  2. Move bottom ring (ring 3): A → C. That's 1 move.
  3. Move 2 rings from B → C: again $T(2) = 3$ moves.
Total: $3 + 1 + 3 = 7$. $T(3) = 7$.

2.2 Notice the pattern

$n$12345678
$T(n)$137153163127255

Each value is one less than a power of 2. In fact:

PATTERN
$T(n) = 2^n - 1$
$T(8) = 2^8 - 1 = 256 - 1 = \boxed{255 \text{ moves}}$

3. Problem 2 — Conjecture for $n$ Rings + Justification

Make a conjecture about the minimum number of moves if you start with $n$ rings. Justify your result.

3.1 The conjecture

CONJECTURE
$T(n) = 2^n - 1$   for all integers $n \ge 1$

3.2 Step 1 — Set up the recurrence

To move $n$ rings from tower A to tower C, the bottom (largest) ring must be moved at least once, and to do so the other $(n-1)$ rings have to be sitting on B (out of the way of both A and C).

PHASE 1 Move top $(n-1)$ rings from A → B.   cost: $T(n-1)$
PHASE 2 Move the bottom ring from A → C.   cost: $1$
PHASE 3 Move the $(n-1)$ rings from B → C onto the bottom ring.   cost: $T(n-1)$
RECURRENCE
$T(n) = 2\,T(n-1) + 1$,   with $T(1) = 1$
Why this is the minimum: The bottom ring has to move at least once (Rule III says the stack ends up on a different tower). Each time it moves, the $(n-1)$ rings above it must be on the third peg (Rule II). So the structure $T(n-1) + 1 + T(n-1)$ is unavoidable. You can't do better than $2T(n-1)+1$.

3.3 Step 2 — Solve the recurrence to get the closed form

This is a non-homogeneous linear recurrence (the "+1" makes it non-homogeneous). We'll use the trick from Phase 3 of your study notes — the shift trick.

TRICK Add 1 to both sides: $$T(n) + 1 = 2T(n-1) + 1 + 1 = 2\bigl(T(n-1) + 1\bigr)$$
SUBSTITUTE Let $U(n) = T(n) + 1$. Then: $$U(n) = 2\,U(n-1), \qquad U(1) = T(1)+1 = 2$$
SOLVE This is a pure geometric recurrence: $U(n) = 2^n$.
UN-SUBSTITUTE $$T(n) = U(n) - 1 = 2^n - 1 \quad \blacksquare$$

3.4 Step 3 — Independent proof by induction

Base case: $n=1$: $T(1) = 1 = 2^1 - 1$. ✓

Inductive step: Assume $T(k) = 2^k - 1$ for some $k \ge 1$. Then:
$\quad T(k+1) = 2T(k) + 1 = 2(2^k - 1) + 1 = 2^{k+1} - 2 + 1 = 2^{k+1} - 1$. ✓

By induction, $T(n) = 2^n - 1$ for all $n \ge 1$. $\blacksquare$
Minimum moves for $n$ rings: $T(n) = 2^n - 1$

4. Problem 3 — Adjacent-Only Constraint

Suppose we change the rules so that you can only move a ring to an adjacent tower, and your objective is still to move the entire stack from the far-left tower to the far-right tower. Find a recursive formula for this more laborious sequence.

4.1 Setup

Three towers in a line: $\;A — B — C$.

A (far-left)
B (middle)
C (far-right)

New rule: A ↔ B and B ↔ C are allowed, but A ↔ C is not a single move (they are not adjacent).

Let $A(n)$ = minimum number of moves to relocate $n$ rings from tower A to tower C under this constraint.

4.2 Tiny case — $n=1$

A(1) One ring on A. To get it to C, you must go through B.
  1. Move the ring: A → B (1 move)
  2. Move the ring: B → C (1 move)
$A(1) = 2$.

4.3 Building the recurrence

Think about how the bottom (largest) ring travels from A to C. Since A and C are not adjacent, the bottom ring must go A → B → C, which takes two moves. Between (and around) those two moves, the other $(n-1)$ rings have to clear the way.

PHASE 1 Move the top $(n-1)$ rings off A so the bottom ring can move A → B. Where do they go? They can't sit on B (B is the bottom ring's next stop) — so they have to go all the way to C.  cost: $A(n-1)$
PHASE 2 Move the bottom ring: A → B.   cost: 1
PHASE 3 Now the bottom ring sits on B. Before it can move B → C, the $(n-1)$ rings on C must get out of the way (you can't drop the bottom ring onto rings smaller than it). Move them from C → A.  cost: $A(n-1)$
PHASE 4 Move the bottom ring: B → C.   cost: 1
PHASE 5 Finally move the $(n-1)$ rings from A → C onto the bottom ring.  cost: $A(n-1)$

Adding up the five phases:

RECURRENCE
$A(n) = 3\,A(n-1) + 2, \quad A(1) = 2$

Predicted: $A(2) = 3\cdot 2 + 2 = 8$.

4.5 Closed form (for full credit / curiosity)

The recurrence $A(n) = 3A(n-1) + 2$ is non-homogeneous. Same shift trick as Problem 2 — but now the multiplier is 3, so we add a different constant.

TRICK Find the constant $c$ so that $A(n) + c = 3(A(n-1) + c)$ becomes pure geometric. Expand: $A(n) + c = 3A(n-1) + 3c$, so $A(n) = 3A(n-1) + 2c$. Compare with $A(n) = 3A(n-1) + 2$: we need $2c = 2$, i.e. $c = 1$.
SUBSTITUTE $V(n) = A(n) + 1$. Then $V(n) = 3V(n-1)$, $\;V(1) = A(1)+1 = 3$.
SOLVE $V(n) = 3^n$.
UN-SUBSTITUTE $A(n) = V(n) - 1 = 3^n - 1$.
$n$12345
$A(n) = 3^n - 1$282680242
Recursive: $A(n) = 3A(n-1) + 2$, $\;A(1) = 2$
Closed form: $A(n) = 3^n - 1$
Notice: the standard Hanoi grows like $2^n$; the adjacent-only Hanoi grows like $3^n$. Same rings, much more painful — that's where "more laborious" comes from.

5. Pre-Calc Connection — Why This Matters for the Test

🔑 Both problems are non-homogeneous linear recurrences.

Both Hanoi recurrences have the same shape:

$T(n) = a \cdot T(n-1) + b$

This is the same family that the test will hit you with via the characteristic-equation method. Let me show you the bridge.

5.1 The shift trick (what we just used twice)

For $T(n) = aT(n-1) + b$, look for a particular solution that's a constant: $T^*(n) = k$ for all $n$.

Plug in: $k = ak + b \;\Longrightarrow\; k(1-a) = b \;\Longrightarrow\; k = \dfrac{b}{1-a}$   (valid when $a \ne 1$).

Then $T(n) - k$ satisfies the homogeneous recurrence $\;u(n) = a\,u(n-1)\;$, which is just $u(n) = u(1) \cdot a^{n-1}$.

So $\;\boxed{T(n) = C \cdot a^n + k}\;$ where $C$ is fixed by the initial condition.

Apply this template:

Problem$a$$b$$k = \dfrac{b}{1-a}$Closed form
Standard Hanoi21$\frac{1}{1-2} = -1$$T(n) = 2^n - 1$
Adjacent-only Hanoi32$\frac{2}{1-3} = -1$$A(n) = 3^n - 1$

5.2 Connect to the characteristic equation

For the homogeneous piece $u(n) = a\cdot u(n-1)$, the characteristic equation is

$x - a = 0 \;\Longrightarrow\; x = a$

so $u(n) = C \cdot a^n$. The general solution to a non-homogeneous recurrence is:

TEMPLATE
$T(n) = \underbrace{C \cdot a^n}_{\text{homogeneous part}} + \underbrace{k}_{\text{particular solution (constant)}}$
How this generalizes: If your recurrence is degree-2 (e.g., Fibonacci-type $L_n = L_{n-1} + L_{n-2}$), the characteristic equation is quadratic ($x^2 - x - 1 = 0$ for Fibonacci), and the homogeneous solution becomes $C_1 r_1^n + C_2 r_2^n$. Same idea, more roots.

💡 One-line takeaway

A recursive formula defines a sequence; the closed form computes it. Hanoi gives you a clean physical reason why each recurrence has that exact shape — use that intuition on the test when the problem is wrapped in a different story (Lego stacking, walkway tilings, etc.).
