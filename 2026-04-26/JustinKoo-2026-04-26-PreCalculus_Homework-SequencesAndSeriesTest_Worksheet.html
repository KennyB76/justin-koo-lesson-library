Sequences & Series Test Prep:
Series Sums, Recursion & Characteristic Equation

Student: Justin Koo Date: 2026-04-26
Standard: Pre-Calculus / Discrete Math — Arithmetic & geometric series, recursive sequences, recursive-to-explicit via characteristic equation, non-homogeneous handling.
Quick Reference
  • Arithmetic sum: \(S_n = \dfrac{n(a_1 + a_n)}{2}\)
  • Geometric sum: \(S_n = g_0 \cdot \dfrac{r^{n} - 1}{r - 1}\)   (\(r \neq 1\))
  • Characteristic equation: \(x^{2} - a_1 x - a_2 = 0\) for \(T_n = a_1 T_{n-1} + a_2 T_{n-2}\)
  • Two distinct roots: \(T_n = c_1 r_1^{\,n} + c_2 r_2^{\,n}\)   Repeated root: \(T_n = (c_1 + c_2 n) r^{n}\)
  • Non-homogeneous: Find particular constant \(K\), let \(b_n = a_n - K\), solve homogeneous \(b_n\), back-substitute.
Part A — Series Sums & Explicit Rules
Directions: Use the appropriate sum formula or solve a system to find the explicit rule. Show all work.
1. Find \(\displaystyle\sum_{i=1}^{15}(2i + 1)\).
2. Find \(\displaystyle\sum_{k=1}^{4} 5 \cdot 3^{\,k-1}\).
3. An arithmetic sequence has \(a_3 = 10\) and \(a_8 = 25\). Write the explicit rule \(a_n\).
Part B — Recursion Problems
Directions: Compute the requested term using the given recurrence. You may iterate or use a known closed form when noted.
4. A path of length \(n\) is built from \(1\)-unit and \(2\)-unit blocks. The number of arrangements satisfies \(L_n = L_{n-1} + L_{n-2}\) with \(L_1 = 1,\; L_2 = 2\). Find \(L_8\).
5. The minimum number of moves for the Tower of Hanoi with \(n\) disks satisfies \(T_n = 2T_{n-1} + 1\) with \(T_1 = 1\). Find \(T_6\) using the closed form \(T_n = 2^{n} - 1\).
6. A sequence satisfies \(a_n = a_{n-1} + a_{n-2}\) with \(a_1 = 1,\; a_2 = 3\). Find \(a_6\).
Part C — Characteristic Equation ⭐ (Test Core)
Directions: Apply the 5-step procedure: (1) identify, (2) char. equation, (3) two-root or repeated form, (4) solve for \(c_1, c_2\), (5) verify.
7. Find the explicit rule for \(T_n = 5T_{n-1} - 6T_{n-2}\) with \(T_0 = 2,\; T_1 = 5\).
8. Find the explicit rule for \(a_n = 6a_{n-1} - 9a_{n-2}\) with \(a_0 = 1,\; a_1 = 6\). (Watch for repeated root.)
9. Find the explicit rule for \(T_n = 3T_{n-1} - 2T_{n-2}\) with \(T_0 = 0,\; T_1 = 1\).
10. Find the explicit rule for \(b_n = b_{n-1} + 2b_{n-2}\) with \(b_0 = 0,\; b_1 = 1\). (Negative root appears.)
Part D — Non-homogeneous Recursion
Directions: Find the particular constant \(K\), substitute \(b_n = a_n - K\) to homogenize, solve, then back-substitute.
11. Find the explicit rule for \(a_n = 2a_{n-1} + 3\) with \(a_0 = 0\).
12. Find the explicit rule for \(T_n = 3T_{n-1} - 4\) with \(T_0 = 5\).
