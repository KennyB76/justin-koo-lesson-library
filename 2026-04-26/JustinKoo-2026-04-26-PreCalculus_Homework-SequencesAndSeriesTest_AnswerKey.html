Part A — Series Sums & Explicit Rules
Problem 1
\(\displaystyle\sum_{i=1}^{15}(2i+1)\)
Arithmetic Sum
First term: \(a_1 = 2(1)+1 = 3\). Last term: \(a_{15} = 2(15)+1 = 31\).
\(S_{15} = \dfrac{15(3+31)}{2} = \dfrac{15 \cdot 34}{2} = 15 \cdot 17 = 255\).
\(S_{15} = 255\)
Problem 2
\(\displaystyle\sum_{k=1}^{4} 5 \cdot 3^{\,k-1}\)
Geometric Sum
Identify: \(g_0 = 5,\; r = 3,\; n = 4\).
\(S_4 = 5 \cdot \dfrac{3^{4} - 1}{3 - 1} = 5 \cdot \dfrac{81 - 1}{2} = 5 \cdot \dfrac{80}{2} = 5 \cdot 40 = 200\).
Check by adding: \(5 + 15 + 45 + 135 = 200\). ✓
\(S_4 = 200\)
Problem 3
Arithmetic with \(a_3 = 10,\; a_8 = 25\). Find \(a_n\).
System of 2 Equations
Equations: \(a_1 + 2d = 10\) and \(a_1 + 7d = 25\).
Subtract: \(5d = 15 \Rightarrow d = 3\).
Plug back: \(a_1 = 10 - 2(3) = 4\).
\(a_n = 4 + 3(n-1) = 3n + 1\).
\(a_n = 3n + 1\)
Part B — Recursion Problems
Problem 4 — Lego
\(L_n = L_{n-1} + L_{n-2}\), \(L_1 = 1, L_2 = 2\). Find \(L_8\).
Fibonacci-type
Iterate forward:
\(L_3 = 3,\;\; L_4 = 5,\;\; L_5 = 8,\;\; L_6 = 13,\;\; L_7 = 21,\;\; L_8 = 21 + 13 = 34\).
\(L_8 = 34\)
Problem 5 — Hanoi
\(T_6\) using \(T_n = 2^{n} - 1\)?
Closed form
\(T_6 = 2^{6} - 1 = 64 - 1 = 63\).
\(T_6 = 63\)
Problem 6
\(a_n = a_{n-1} + a_{n-2}\), \(a_1 = 1, a_2 = 3\). Find \(a_6\).
Iterate
\(a_3 = 3 + 1 = 4,\;\; a_4 = 4 + 3 = 7,\;\; a_5 = 7 + 4 = 11,\;\; a_6 = 11 + 7 = 18\).
\(a_6 = 18\)
Part C — Characteristic Equation ⭐
Problem 7
\(T_n = 5T_{n-1} - 6T_{n-2}\), \(T_0 = 2, T_1 = 5\). Find \(T_n\).
Two distinct roots
Step 2. Char. equation: \(x^{2} - 5x + 6 = 0 \Rightarrow (x-2)(x-3) = 0\). Roots: \(r_1 = 3,\; r_2 = 2\).
Step 3. \(T_n = c_1 \cdot 3^{n} + c_2 \cdot 2^{n}\).
Step 4. \(n=0\): \(c_1 + c_2 = 2\). \(\;n=1\): \(3c_1 + 2c_2 = 5\). Solve: \(c_1 = 1,\; c_2 = 1\).
Step 5 (check). \(T_2\) recurrence: \(5(5) - 6(2) = 13\). Closed form: \(9 + 4 = 13\). ✓
\(T_n = 3^{n} + 2^{n}\)
Problem 8
\(a_n = 6a_{n-1} - 9a_{n-2}\), \(a_0 = 1, a_1 = 6\). Find \(a_n\).
Repeated root ⚠ \(n\) factor
Step 2. \(x^{2} - 6x + 9 = 0 \Rightarrow (x-3)^{2} = 0\). Repeated root \(r = 3\).
Step 3. Repeated form: \(a_n = (c_1 + c_2 n) \cdot 3^{n}\).
Step 4. \(n=0\): \(c_1 = 1\). \(\;n=1\): \((1 + c_2)(3) = 6 \Rightarrow c_2 = 1\).
Step 5 (check). \(a_2\) recurrence: \(6(6) - 9(1) = 27\). Closed form: \((1+2)(9) = 27\). ✓
\(a_n = (1 + n) \cdot 3^{n}\)
Problem 9
\(T_n = 3T_{n-1} - 2T_{n-2}\), \(T_0 = 0, T_1 = 1\). Find \(T_n\).
Two distinct roots Root \(r=1\)
Step 2. \(x^{2} - 3x + 2 = 0 \Rightarrow (x-1)(x-2) = 0\). Roots: \(1, 2\).
Step 3. \(T_n = c_1 \cdot 1^{n} + c_2 \cdot 2^{n} = c_1 + c_2 \cdot 2^{n}\).
Step 4. \(n=0\): \(c_1 + c_2 = 0\). \(\;n=1\): \(c_1 + 2c_2 = 1\). Subtract: \(c_2 = 1,\; c_1 = -1\).
Step 5 (check). \(T_2\) recurrence: \(3(1) - 2(0) = 3\). Closed form: \(-1 + 4 = 3\). ✓
\(T_n = 2^{n} - 1\)
Problem 10
\(b_n = b_{n-1} + 2b_{n-2}\), \(b_0 = 0, b_1 = 1\). Find \(b_n\).
Negative root Fractional \(c\)
Step 2. \(x^{2} - x - 2 = 0 \Rightarrow (x-2)(x+1) = 0\). Roots: \(2, -1\).
Step 3. \(b_n = c_1 \cdot 2^{n} + c_2 \cdot (-1)^{n}\).
Step 4. \(n=0\): \(c_1 + c_2 = 0\). \(\;n=1\): \(2c_1 - c_2 = 1\). Solve: \(c_1 = \tfrac{1}{3},\; c_2 = -\tfrac{1}{3}\).
Step 5 (check). \(b_2\) recurrence: \(1 + 2(0) = 1\). Closed form: \(\tfrac{1}{3}(4 - 1) = 1\). ✓
\(b_n = \dfrac{1}{3}\bigl(2^{n} - (-1)^{n}\bigr)\)
Part D — Non-homogeneous Recursion
Problem 11
\(a_n = 2a_{n-1} + 3\), \(a_0 = 0\). Find \(a_n\).
Particular solution Substitution
Step 1 (particular). Assume \(a_n = K\) constant: \(K = 2K + 3 \Rightarrow K = -3\).
Step 2 (substitute). Let \(b_n = a_n - K = a_n + 3\). Then:
\(b_n = a_n + 3 = (2a_{n-1} + 3) + 3 = 2a_{n-1} + 6 = 2(a_{n-1} + 3) = 2 b_{n-1}\). Homogeneous!
Step 3. \(b_0 = a_0 + 3 = 3\), and \(b_n = 2 b_{n-1}\) gives \(b_n = 3 \cdot 2^{n}\).
Step 4 (back-substitute). \(a_n = b_n - 3 = 3 \cdot 2^{n} - 3\).
Check. \(a_1\) recurrence: \(2(0) + 3 = 3\). Closed form: \(6 - 3 = 3\). ✓
\(a_n = 3 \cdot 2^{n} - 3\)
Problem 12
\(T_n = 3T_{n-1} - 4\), \(T_0 = 5\). Find \(T_n\).
Particular solution Substitution
Step 1 (particular). \(K = 3K - 4 \Rightarrow 2K = 4 \Rightarrow K = 2\).
Step 2 (substitute). Let \(b_n = T_n - 2\). Then:
\(b_n = T_n - 2 = (3T_{n-1} - 4) - 2 = 3T_{n-1} - 6 = 3(T_{n-1} - 2) = 3 b_{n-1}\). Homogeneous!
Step 3. \(b_0 = T_0 - 2 = 3\), and \(b_n = 3 b_{n-1}\) gives \(b_n = 3 \cdot 3^{n} = 3^{n+1}\).
Step 4 (back-substitute). \(T_n = b_n + 2 = 3^{n+1} + 2\).
Check. \(T_1\) recurrence: \(3(5) - 4 = 11\). Closed form: \(9 + 2 = 11\). ✓
\(T_n = 3^{\,n+1} + 2\)
