📐 Phase 1 — Arithmetic & Geometric Series

1. Arithmetic Sequence (등차수열)

각 항이 앞 항보다 일정한 차이 \(d\) 만큼 더해지는 수열.

Arithmetic — Explicit & Sum

Explicit: \(a_n = a_1 + (n-1)d\)

Sum: \(S_n = \dfrac{n(a_1 + a_n)}{2}\)

2. Geometric Sequence (등비수열)

각 항이 앞 항에 일정한 비율 \(r\) 만큼 곱해지는 수열.

Geometric — Explicit & Sum

Explicit: \(g_n = g_0 \cdot r^{n}\)   (또는 \(a_n = a_1 \cdot r^{\,n-1}\))

Sum: \(S_n = g_0 \cdot \dfrac{r^{n} - 1}{r - 1}\)   (단 \(r \neq 1\))

Example 1 — Arithmetic Sum
Find \(\sum_{i=1}^{20}(3i + 5)\).

Step 1. 첫 항 \(a_1 = 3(1)+5 = 8\), 마지막 항 \(a_{20} = 3(20)+5 = 65\).

Step 2. 등차수열의 합 공식: \(S_{20} = \dfrac{20(8 + 65)}{2} = 10 \cdot 73 = 730\).

\(S_{20} = 730\)

Example 2 — Geometric Sum
Find \(\sum_{k=1}^{6} 3 \cdot 2^{k-1}\).

Step 1. Identify: \(g_0 = 3,\; r = 2,\; n = 6\).

Step 2. 공식 적용: \(S_6 = 3 \cdot \dfrac{2^{6} - 1}{2 - 1} = 3 \cdot \dfrac{63}{1} = 189\).

검산: \(3 + 6 + 12 + 24 + 48 + 96 = 189\). ✓

\(S_6 = 189\)

🔄 Phase 2 — Recursion Problems

점화식 (recursive definition) 으로 정의된 문제를 알아보고 패턴을 찾는다. 이게 Phase 3 (특성방정식) 의 입력이 된다.

1. Lego / Fibonacci 패턴

1 또는 2 길이의 블록으로 길이 \(n\) 의 길을 만드는 가짓수.

Lego (Fibonacci)

\(L_n = L_{n-1} + L_{n-2}\),   \(L_1 = 1,\; L_2 = 2\)

시퀀스: 1, 2, 3, 5, 8, 13, 21, ...

2. Tower of Hanoi

\(n\) 개 디스크를 옮기는 데 필요한 최소 이동 횟수.

Hanoi — 비동차 점화식

\(T_n = 2T_{n-1} + 1\),   \(T_1 = 1\)

닫힌 형태: \(T_n = 2^{n} - 1\)

주의: "+1" 때문에 비동차. Phase 4 에서 해결.

3. Walkway / No-Adjacent-AA 패턴

두 글자 (A, B) 로 길이 \(n\) 짜리 문자열을 만들 때 "AA" 가 안 나오게 하는 가짓수.

Walkway

\(W_n = W_{n-1} + W_{n-2}\)   (Lego 와 같은 구조)

Example 3 — Lego 직접 계산
길이 7 의 길을 1, 2 길이 블록으로 만드는 가짓수 \(L_7\) 은?

Step 1. 점화식: \(L_n = L_{n-1} + L_{n-2}\), \(L_1 = 1, L_2 = 2\).

Step 2. 차근차근 계산:

\(L_3 = L_2 + L_1 = 2 + 1 = 3\)
\(L_4 = L_3 + L_2 = 3 + 2 = 5\)
\(L_5 = L_4 + L_3 = 5 + 3 = 8\)
\(L_6 = L_5 + L_4 = 8 + 5 = 13\)
\(L_7 = L_6 + L_5 = 13 + 8 = 21\)

\(L_7 = 21\)

Example 4 — Hanoi 직접 계산
디스크 5개의 Hanoi 이동 횟수 \(T_5\) 는?

방법 A (점화식): \(T_1 = 1,\; T_n = 2T_{n-1}+1\).

\(T_2 = 2(1)+1 = 3\)
\(T_3 = 2(3)+1 = 7\)
\(T_4 = 2(7)+1 = 15\)
\(T_5 = 2(15)+1 = 31\)

방법 B (닫힌 형태): \(T_5 = 2^{5} - 1 = 32 - 1 = 31\). ✓

\(T_5 = 31\)

⭐ Phase 3 — Characteristic Equation (시험 핵심)

Recursive 정의를 Explicit (닫힌 형태) 로 변환하는 표준 절차. 시험에 100% 나옴.

⭐ 5단계 절차 — 외워야 함
  1. 점화식 \(T_n = a_1 T_{n-1} + a_2 T_{n-2}\) 형태 확인 (동차 only).
  2. 특성방정식 \(x^{2} - a_1 x - a_2 = 0\) 풀어 두 근 \(r_1, r_2\) 구함.
  3. 두 근 다름: \(T_n = c_1 r_1^{n} + c_2 r_2^{n}\).
    중근: \(T_n = (c_1 + c_2 n) \cdot r^{n}\) ⚠️ n 곱하는 거 함정.
  4. \(n=0,\; n=1\) 대입한 2×2 시스템 풀어 \(c_1, c_2\) 결정.
  5. 닫힌 형태 + 검산 (예: \(T_2\) 점화식 vs 닫힌 형태 비교).
Example 5 — Fibonacci 닫힌 형태
\(F_n = F_{n-1} + F_{n-2}\), \(F_0 = 0, F_1 = 1\). \(F_n\) 의 닫힌 형태를 구하라. (Binet's formula)

Step 1. 형태 확인: \(a_1 = 1, a_2 = 1\).

Step 2. 특성방정식: \(x^{2} - x - 1 = 0\).

근: \(x = \dfrac{1 \pm \sqrt{1 + 4}}{2} = \dfrac{1 \pm \sqrt{5}}{2}\).

\(r_1 = \varphi = \dfrac{1 + \sqrt{5}}{2}\), \(r_2 = \psi = \dfrac{1 - \sqrt{5}}{2}\).

Step 3. 두 근 다름: \(F_n = c_1 \varphi^{n} + c_2 \psi^{n}\).

Step 4. 초기조건 대입:

\(n=0\): \(c_1 + c_2 = 0\) → \(c_2 = -c_1\).

\(n=1\): \(c_1 \varphi + c_2 \psi = 1\) → \(c_1(\varphi - \psi) = 1\) → \(c_1 = \dfrac{1}{\sqrt{5}}\).

Step 5. 닫힌 형태:

\(F_n = \dfrac{1}{\sqrt{5}}\!\left(\varphi^{n} - \psi^{n}\right)\)

Example 6 — 중근 케이스 (n 곱하는 함정)
\(T_n = 4T_{n-1} - 4T_{n-2}\), \(T_0 = 1, T_1 = 6\). \(T_n\) 닫힌 형태?

Step 1. \(a_1 = 4, a_2 = -4\).

Step 2. 특성방정식: \(x^{2} - 4x + 4 = 0\) → \((x-2)^{2} = 0\) → 중근 \(r = 2\).

Step 3. 중근 형태: \(T_n = (c_1 + c_2 n) \cdot 2^{n}\).

Step 4. 초기조건:

\(n=0\): \(T_0 = c_1 \cdot 1 = 1\) → \(c_1 = 1\).

\(n=1\): \(T_1 = (1 + c_2)(2) = 6\) → \(1 + c_2 = 3\) → \(c_2 = 2\).

Step 5. 닫힌 형태:

\(T_n = (1 + 2n) \cdot 2^{n}\)

검산 \(T_2 = (1 + 4)(4) = 20\). 점화식: \(T_2 = 4(6) - 4(1) = 24 - 4 = 20\). ✓

🌶️ Phase 4 — Non-homogeneous Recursion

표준 특성방정식 절차는 동차 (homogeneous) 점화식 전용 — 모든 항이 \(T_*\) 들로만 구성. +1 같은 상수항 이 끼면 비동차이고, 한 단계 변환 이 필요하다.

핵심 아이디어 — 특해 빼서 동차화

비동차 점화식의 특수해 (particular solution) 를 찾고, 그걸 빼서 새 변수를 만들면 동차가 된다.

Hanoi 의 비동차 처리

원래 점화식: \(H_n = 2H_{n-1} + 1\)

변환: 양변에 \(+1\) 더하면 \(H_n + 1 = 2(H_{n-1} + 1)\).

\(U_n := H_n + 1\) 로 두면 → \(U_n = 2U_{n-1}\) (동차!).

초기조건 \(U_1 = H_1 + 1 = 1 + 1 = 2\) → \(U_n = 2^{n}\).

되돌리면 → \(\boxed{H_n = 2^{n} - 1}\)

Example 7 — 비동차 절차 직접
\(a_n = 3a_{n-1} + 2\), \(a_0 = 1\) 의 닫힌 형태를 구하라.

Step 1. 특해 찾기: \(a_n\) 이 어떤 상수 \(K\) 라고 가정 → \(K = 3K + 2\) → \(K = -1\).

Step 2. 변환: \(b_n := a_n - (-1) = a_n + 1\). 그러면:

\(b_n = a_n + 1 = 3a_{n-1} + 2 + 1 = 3(a_{n-1} + 1) = 3 b_{n-1}\) (동차!).

Step 3. 초기: \(b_0 = a_0 + 1 = 2\). → \(b_n = 2 \cdot 3^{n}\).

Step 4. 되돌리기: \(a_n = b_n - 1 = 2 \cdot 3^{n} - 1\).

\(a_n = 2 \cdot 3^{n} - 1\)

검산 \(a_1 = 2(3) - 1 = 5\). 점화식: \(a_1 = 3(1) + 2 = 5\). ✓

✅ 시험 직전 체크리스트

각 항목 머릿속으로 답할 수 있으면 ✓. 못 하면 그 phase 로 돌아가서 복습.

  • Arithmetic 와 Geometric 시퀀스 — explicit, recursive 둘 다 쓸 수 있는가?
  • \(S_n\) 공식 두 개 (arithmetic, geometric) 외웠는가?
  • Difference table — Δ가 상수가 되는 단계 = 차수, 안다.
  • Lego = Fibonacci, Hanoi = \(2^{n}-1\) 즉답 가능한가?
  • 특성방정식 5단계 막힘없이 적용 가능한가?
  • 중근일 때 \((c_1 + c_2 n) r^{n}\) 형태 — \(n\) 곱하는 거 잊지 않는다.
  • 비동차 점화식 — 특해 빼서 동차화하는 절차 안다.
  • 검산 — 닫힌 형태에 \(n=2\) 넣은 값과 점화식 결과 일치하는지 확인하는 습관.
💡 마지막 팁: 점화식 보면 자동으로 "동차인가? 비동차인가?" 묻는 습관. 비동차면 특해부터, 동차면 곧장 특성방정식.
