Problem 1 — Sum of First n Positive Integers
Prove: $\displaystyle 1 + 2 + \cdots + n = \dfrac{n(n+1)}{2}$ for all $n \ge 1$.
Base Case Inductive Step Sum Formula
Base Case ($n = 1$): LHS $= 1$.   RHS $= \dfrac{1 \cdot 2}{2} = 1$.   $\checkmark$
Inductive Hypothesis: Assume $1 + 2 + \cdots + k = \dfrac{k(k+1)}{2}$ for some $k \ge 1$.
Inductive Step (show true for $n = k+1$):
$1 + 2 + \cdots + k + (k+1)$
$= \dfrac{k(k+1)}{2} + (k+1)$    (by inductive hypothesis)
$= (k+1)\!\left(\dfrac{k}{2} + 1\right)$
$= (k+1) \cdot \dfrac{k+2}{2}$
$= \dfrac{(k+1)(k+2)}{2}$    $\checkmark$   (this is $\tfrac{(k+1)((k+1)+1)}{2}$)
Conclusion: By the Principle of Mathematical Induction, the formula holds for all $n \ge 1$.
$\displaystyle\sum_{i=1}^{n} i = \dfrac{n(n+1)}{2}$   $\blacksquare$
Problem 2 — Sum of Squares
Prove: $\displaystyle 1^2 + 2^2 + \cdots + n^2 = \dfrac{n(n+1)(2n+1)}{6}$ for all $n \ge 1$.
Base Case Inductive Step Factoring
Base Case ($n=1$): LHS $= 1$.   RHS $= \dfrac{1 \cdot 2 \cdot 3}{6} = 1$.   $\checkmark$
Inductive Hypothesis: Assume $1^2 + \cdots + k^2 = \dfrac{k(k+1)(2k+1)}{6}$.
Inductive Step:
$1^2 + \cdots + k^2 + (k+1)^2$
$= \dfrac{k(k+1)(2k+1)}{6} + (k+1)^2$    (by IH)
$= \dfrac{k(k+1)(2k+1) + 6(k+1)^2}{6}$
$= \dfrac{(k+1)\bigl[k(2k+1) + 6(k+1)\bigr]}{6}$
$= \dfrac{(k+1)(2k^2 + 7k + 6)}{6}$
$= \dfrac{(k+1)(k+2)(2k+3)}{6}$    (factor quadratic)
$= \dfrac{(k+1)\bigl((k+1)+1\bigr)\bigl(2(k+1)+1\bigr)}{6}$    $\checkmark$
Conclusion: Formula holds for all $n \ge 1$ by induction.
$\displaystyle\sum_{i=1}^{n} i^2 = \dfrac{n(n+1)(2n+1)}{6}$   $\blacksquare$
Problem 3 — Geometric Series Sum
Prove: $\displaystyle 1 + r + r^2 + \cdots + r^{n-1} = \dfrac{r^n - 1}{r-1}$ for all $n \ge 1$, $r \ne 1$.
Geometric Series Base Case Inductive Step
Base Case ($n=1$): LHS $= r^0 = 1$.   RHS $= \dfrac{r^1 - 1}{r-1} = 1$.   $\checkmark$
Inductive Hypothesis: Assume $1 + r + \cdots + r^{k-1} = \dfrac{r^k-1}{r-1}$.
Inductive Step:
$1 + r + \cdots + r^{k-1} + r^k$
$= \dfrac{r^k - 1}{r-1} + r^k$    (by IH)
$= \dfrac{r^k - 1 + r^k(r-1)}{r-1}$
$= \dfrac{r^k - 1 + r^{k+1} - r^k}{r-1}$
$= \dfrac{r^{k+1} - 1}{r-1}$    $\checkmark$
Conclusion: The geometric series formula holds for all $n \ge 1$.
$\displaystyle\sum_{i=0}^{n-1} r^i = \dfrac{r^n - 1}{r-1}$   $\blacksquare$
Problem 4 — Divisibility by 6
Prove: $6 \mid (n^3 - n)$ for all integers $n \ge 1$.
Divisibility Factoring Inductive Step
Base Case ($n=1$): $1^3 - 1 = 0 = 6 \cdot 0$. Divisible by 6.   $\checkmark$
Inductive Hypothesis: Assume $6 \mid (k^3 - k)$, i.e., $k^3 - k = 6m$ for some integer $m$.
Inductive Step:
$(k+1)^3 - (k+1)$
$= k^3 + 3k^2 + 3k + 1 - k - 1$
$= (k^3 - k) + 3k^2 + 3k$
$= (k^3 - k) + 3k(k+1)$
Now, $k^3 - k = 6m$ by IH. Also, $k(k+1)$ is a product of two consecutive integers, so it is always even: $k(k+1) = 2t$ for some integer $t$.
Therefore: $(k+1)^3 - (k+1) = 6m + 3 \cdot 2t = 6m + 6t = 6(m+t)$, which is divisible by 6.   $\checkmark$
Conclusion: $6 \mid (n^3 - n)$ for all $n \ge 1$.
Alternate check: $n^3 - n = n(n-1)(n+1)$ = product of three consecutive integers, always divisible by $3! = 6$.
$6 \mid (n^3 - n)$ for all $n \ge 1$   $\blacksquare$
Problem 5 — Inequality: $2^n \ge n^2$ for $n \ge 4$
Prove: $2^n \ge n^2$ for all integers $n \ge 4$.
Inequality Non-standard Base Case Exponential vs Polynomial
Why not $n=1,2,3$: $2^1=2 < 1^2$ — wait, actually $2 > 1$. Let's recheck: $n=2$: $2^2=4 = 2^2=4$ ✓. $n=3$: $2^3=8 < 9=3^2$ ✗. So $n=3$ fails; base case must start at $n=4$.
Base Case ($n=4$): $2^4 = 16 = 4^2 = 16$.   $16 \ge 16$.   $\checkmark$
Inductive Hypothesis: Assume $2^k \ge k^2$ for some $k \ge 4$.
Inductive Step (show $2^{k+1} \ge (k+1)^2$):
$2^{k+1} = 2 \cdot 2^k \ge 2k^2$    (by IH, multiplying both sides by 2)
Need to show: $2k^2 \ge (k+1)^2 = k^2 + 2k + 1$
i.e., $k^2 - 2k - 1 \ge 0$
i.e., $(k-1)^2 \ge 2$
For $k \ge 4$: $(k-1) \ge 3$, so $(k-1)^2 \ge 9 \ge 2$.   $\checkmark$
Chaining: $2^{k+1} \ge 2k^2 \ge (k+1)^2$.   $\checkmark$
Conclusion: $2^n \ge n^2$ for all integers $n \ge 4$.
$2^n \ge n^2$ for all $n \ge 4$   $\blacksquare$
