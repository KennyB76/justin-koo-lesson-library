Problem 1 — Sum of Cubes: $\displaystyle\sum_{i=1}^{n} i^3 = \left(\frac{n(n+1)}{2}\right)^2$
Base Case ($n=1$): LHS $= 1^3 = 1$.   RHS $= \left(\frac{1\cdot2}{2}\right)^2 = 1^2 = 1$.   $1 = 1$ $\checkmark$
Inductive Hyp.: Assume $\displaystyle\sum_{i=1}^{k} i^3 = \left(\frac{k(k+1)}{2}\right)^2$ for some $k \ge 1$.
Inductive Step: $\displaystyle\sum_{i=1}^{k+1} i^3 = \left(\frac{k(k+1)}{2}\right)^2 + (k+1)^3$

$= \frac{k^2(k+1)^2}{4} + (k+1)^3$

$= (k+1)^2\left[\frac{k^2}{4} + (k+1)\right]$

$= (k+1)^2 \cdot \frac{k^2 + 4k + 4}{4}$

$= (k+1)^2 \cdot \frac{(k+2)^2}{4} = \left(\frac{(k+1)(k+2)}{2}\right)^2$   $\checkmark$
Conclusion: By PMI, $\displaystyle\sum_{i=1}^{n} i^3 = \left(\frac{n(n+1)}{2}\right)^2$ for all $n \ge 1$. $\blacksquare$
Problem 2 — Telescoping Sum: $\displaystyle\sum_{i=1}^{n} \frac{1}{i(i+1)} = \frac{n}{n+1}$
Base Case ($n=1$): LHS $= \dfrac{1}{1\cdot2} = \dfrac{1}{2}$.   RHS $= \dfrac{1}{2}$.   $\checkmark$
Inductive Hyp.: Assume $\displaystyle\sum_{i=1}^{k} \frac{1}{i(i+1)} = \frac{k}{k+1}$ for some $k \ge 1$.
Inductive Step: $\displaystyle\sum_{i=1}^{k+1} \frac{1}{i(i+1)} = \frac{k}{k+1} + \frac{1}{(k+1)(k+2)}$

$= \frac{k(k+2) + 1}{(k+1)(k+2)} = \frac{k^2+2k+1}{(k+1)(k+2)} = \frac{(k+1)^2}{(k+1)(k+2)} = \frac{k+1}{k+2}$   $\checkmark$
Conclusion: By PMI, $\displaystyle\sum_{i=1}^{n} \frac{1}{i(i+1)} = \frac{n}{n+1}$ for all $n \ge 1$. $\blacksquare$

Problem 3 — Powers of 3: $1 + 3 + \cdots + 3^{n-1} = \dfrac{3^n-1}{2}$
Base Case ($n=1$): LHS $= 3^0 = 1$.   RHS $= \dfrac{3^1-1}{2} = 1$.   $\checkmark$
Inductive Hyp.: Assume $1 + 3 + \cdots + 3^{k-1} = \dfrac{3^k-1}{2}$ for some $k \ge 1$.
Inductive Step: $\underbrace{1 + 3 + \cdots + 3^{k-1}}_{=\,\frac{3^k-1}{2}} + 3^k$

$= \frac{3^k - 1}{2} + \frac{2 \cdot 3^k}{2} = \frac{3 \cdot 3^k - 1}{2} = \frac{3^{k+1}-1}{2}$   $\checkmark$
Conclusion: By PMI, the formula holds for all $n \ge 1$. $\blacksquare$
Problem 4 — Divisibility: $7 \mid (8^n - 1)$
Base Case ($n=1$): $8^1 - 1 = 7 = 7 \cdot 1$.   Divisible by 7.   $\checkmark$
Inductive Hyp.: Assume $8^k - 1 = 7m$ for some integer $m$.
Inductive Step: $8^{k+1} - 1 = 8 \cdot 8^k - 1 = 8(8^k - 1) + 8 - 1 = 8(8^k-1) + 7$

$= 8 \cdot 7m + 7 = 7(8m + 1)$   $\checkmark$   (since $8m+1 \in \mathbb{Z}$)
Conclusion: By PMI, $7 \mid (8^n-1)$ for all $n \ge 1$. $\blacksquare$
Problem 5 — Inequality: $3^n \ge 2n+1$ for $n \ge 1$
Base Case ($n=1$): $3^1 = 3 \ge 2(1)+1 = 3$.   $3 \ge 3$ $\checkmark$
Inductive Hyp.: Assume $3^k \ge 2k+1$ for some $k \ge 1$.
Inductive Step: $3^{k+1} = 3 \cdot 3^k \overset{\text{I.H.}}{\ge} 3(2k+1) = 6k + 3$

We need $6k + 3 \ge 2(k+1)+1 = 2k+3$.

$6k + 3 \ge 2k + 3 \iff 4k \ge 0$, which holds for all $k \ge 1$.   $\checkmark$
Conclusion: By PMI, $3^n \ge 2n+1$ for all $n \ge 1$. $\blacksquare$
