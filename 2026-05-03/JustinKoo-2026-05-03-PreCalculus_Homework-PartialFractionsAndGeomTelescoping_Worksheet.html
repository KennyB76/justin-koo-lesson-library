Part 1 — Algebra Foundations: Complex Fractions & Partial Fractions
Why this part comes first: every infinite geometric sum lands in the shape $\dfrac{g_0}{1-r}$, which is a complex fraction. And every telescoping sum starts with a partial-fraction split. Master both before moving on. Strategy for complex fractions: simplify inside-out — innermost fraction first, then outward.
Example 1 — Complex fraction with sums in numerator and denominator
Simplify $\;\displaystyle\frac{\frac{1}{3} + \frac{1}{6}}{1 - \frac{1}{4}}$.

Inside-out: compute each layer separately first.

Numerator: $\;\tfrac{1}{3} + \tfrac{1}{6} = \tfrac{2}{6} + \tfrac{1}{6} = \tfrac{3}{6} = \tfrac{1}{2}$.

Denominator: $\;1 - \tfrac{1}{4} = \tfrac{3}{4}$.

Now divide: $\;\dfrac{1/2}{3/4} = \tfrac{1}{2} \cdot \tfrac{4}{3} = \tfrac{4}{6} = \tfrac{2}{3}$.

Answer: $\dfrac{2}{3}$.

Example 2 — Partial fraction decomposition
Find $A,B$ so that $\;\displaystyle\frac{1}{n(n+3)} = \frac{A}{n} + \frac{B}{n+3}$.

Combine the right side over $n(n+3)$:   $\dfrac{A(n+3) + Bn}{n(n+3)} = \dfrac{1}{n(n+3)}$.

Numerators must match for all $n$: $\;A(n+3) + Bn = 1 \;\Rightarrow\; (A+B)n + 3A = 1$.

Match coefficients:  $n$-coeff: $A + B = 0$.   constant: $3A = 1 \Rightarrow A = \tfrac{1}{3}$, so $B = -\tfrac{1}{3}$.

$A = \tfrac{1}{3},\;\; B = -\tfrac{1}{3}$.

1. Simplify $\;\displaystyle\frac{\frac{2}{5} + \frac{1}{4}}{1 - \frac{3}{10}}$.   (Inside-out: numerator and denominator separately first.)
2. Simplify the nested complex fraction $\;\displaystyle\frac{1}{1 - \dfrac{1}{1 + \dfrac{1}{3}}}$.   (Three layers — start at the innermost $1 + \tfrac{1}{3}$ and work your way out.)
3. Find $A, B$ such that $\;\displaystyle \frac{1}{n(n+2)} = \frac{A}{n} + \frac{B}{n+2}$.   (Note the gap of $2$ between factors.)
4. Find $A, B$ such that $\;\displaystyle \frac{1}{(3n-1)(3n+2)} = \frac{A}{3n-1} + \frac{B}{3n+2}$.
5. First factor the denominator, then find $A, B$: $\;\displaystyle\frac{1}{n^2 + 4n + 3} = \frac{A}{?} + \frac{B}{?}$.   (Hint: $n^2 + 4n + 3$ factors as $(n+1)(n+3)$.)
Part 2 — Geometric Series (Finite & Infinite)
Reminder. Finite: $\displaystyle\sum_{k=0}^{n} g_0 r^k = g_0 \cdot \dfrac{1 - r^{\,n+1}}{1 - r}$   (works for any $r \ne 1$).    Infinite: $\displaystyle\sum_{k=0}^{\infty} g_0 r^k = \dfrac{g_0}{1-r}$   only when $|r|<1$ (else write DIVERGES).
Example 1 — Repeating decimal as a geometric series
Express $\;0.\overline{27} = 0.272727\ldots\;$ as an exact fraction.

Split into chunks of two digits: $\;0.\overline{27} = 0.27 + 0.0027 + 0.000027 + \cdots$

This is geometric with first term $g_0 = \tfrac{27}{100}$ and ratio $r = \tfrac{1}{100}$ (each chunk is $1/100$ of the previous).

Since $|r| = 1/100 < 1$: $$S_\infty = \frac{27/100}{1 - 1/100} = \frac{27/100}{99/100} = \frac{27}{99} = \frac{3}{11}$$

Check: $3 \div 11 = 0.272727\ldots$ ✓

Answer: $\;0.\overline{27} = \dfrac{3}{11}$.

This trick works for any repeating decimal — pick a chunk length equal to the repeat block.

Example 2 — Finite geometric sum starting at index $\ne 0$ (use a shift)
Compute $\;\displaystyle\sum_{k=2}^{6} \left(\tfrac{1}{2}\right)^k$.

Method (shift to start at $0$): let $j = k - 2$, so $k = j+2$. Then $(1/2)^k = (1/2)^{j+2} = \tfrac{1}{4}(1/2)^j$. The sum becomes $$\sum_{j=0}^{4} \tfrac{1}{4}(1/2)^j = \tfrac{1}{4} \cdot \frac{1 - (1/2)^5}{1 - 1/2} = \tfrac{1}{4} \cdot \frac{1 - 1/32}{1/2} = \tfrac{1}{4} \cdot 2 \cdot \tfrac{31}{32} = \tfrac{31}{64}$$

Sanity check (direct): $\;\tfrac{1}{4} + \tfrac{1}{8} + \tfrac{1}{16} + \tfrac{1}{32} + \tfrac{1}{64} = \tfrac{16+8+4+2+1}{64} = \tfrac{31}{64}$ ✓

Answer: $\;\dfrac{31}{64}$.

Part 3 — Telescoping Series & Power Series IoC (Final Challenge)
Strategy. Telescoping series come from partial-fraction splits (Part 1). Write $S_N$ explicitly, watch what cancels in pairs, take $N \to \infty$. The very last problem (#5) introduces a power series — same recipe as a geometric series, but the ratio contains $x$, so you must find the values of $x$ where $|r(x)| < 1$ (the IoC).
Example 1 — Basic telescoping
Evaluate $\;\displaystyle\sum_{n=1}^{\infty} \frac{1}{n(n+1)}$.

Partial fractions: $\;\dfrac{1}{n(n+1)} = \dfrac{1}{n} - \dfrac{1}{n+1}$ (verify: $\tfrac{(n+1)-n}{n(n+1)} = \tfrac{1}{n(n+1)}$ ✓).

Build $S_N$ explicitly: $$S_N = \Big(1 - \tfrac{1}{2}\Big) + \Big(\tfrac{1}{2} - \tfrac{1}{3}\Big) + \Big(\tfrac{1}{3} - \tfrac{1}{4}\Big) + \cdots + \Big(\tfrac{1}{N} - \tfrac{1}{N+1}\Big)$$

Every middle piece cancels (the $-\tfrac{1}{2}$ kills the next $+\tfrac{1}{2}$, etc.). Only the very first $1$ and the very last $-\tfrac{1}{N+1}$ survive: $\;S_N = 1 - \tfrac{1}{N+1}$.

Take $N \to \infty$: $\;\tfrac{1}{N+1} \to 0$, so $S_\infty = 1$.

Answer: $\;1$.

Example 2 — Power series with $x$ in the ratio: find IoC and sum function
Find the Interval of Convergence and the sum function $P(x)$ for $\;\displaystyle\sum_{k=0}^{\infty}\left(-\tfrac{2}{x}\right)^k$.

Step 1 — Identify $g_0$ and $r(x)$. At $k=0$, term is $1$, so $g_0 = 1$. Ratio $r(x) = -\tfrac{2}{x}$.

Step 2 — Convergence: $|r(x)| < 1$. $$\left|-\tfrac{2}{x}\right| < 1 \;\Longrightarrow\; \frac{2}{|x|} < 1 \;\Longrightarrow\; |x| > 2 \;\Longrightarrow\; x > 2 \text{ or } x < -2$$ IoC $= (-\infty, -2) \cup (2, \infty)$.

Step 3 — Sum function (clean up the complex fraction!). $$P(x) = \frac{g_0}{1 - r(x)} = \frac{1}{1 - (-2/x)} = \frac{1}{1 + 2/x}$$ Multiply numerator and denominator by $x$ (Part-1 trick): $$P(x) = \frac{1 \cdot x}{(1 + 2/x) \cdot x} = \frac{x}{x + 2}$$

IoC: $(-\infty,-2)\cup(2,\infty)$,   $P(x) = \dfrac{x}{x+2}$.

Step 3 is exactly the inside-out complex-fraction trick from Part 1. Now you see why we drilled it.
