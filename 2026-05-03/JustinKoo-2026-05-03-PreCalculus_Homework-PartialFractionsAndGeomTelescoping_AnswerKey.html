Part 1 — Algebra Foundations
Problem 1
Simplify $\;\dfrac{2/5 + 1/4}{1 - 3/10}$.
complex fraction inside-out

Numerator: $\;\tfrac{2}{5} + \tfrac{1}{4} = \tfrac{8}{20} + \tfrac{5}{20} = \tfrac{13}{20}$.

Denominator: $\;1 - \tfrac{3}{10} = \tfrac{7}{10}$.

Divide: $\;\dfrac{13/20}{7/10} = \tfrac{13}{20} \cdot \tfrac{10}{7} = \tfrac{130}{140} = \tfrac{13}{14}$.

$\dfrac{13}{14}$
Problem 2 — Nested (3 layers)
Simplify $\;\dfrac{1}{1 - \dfrac{1}{1 + \frac{1}{3}}}$.
nested fraction inside-out

Layer 1 (innermost): $\;1 + \tfrac{1}{3} = \tfrac{4}{3}$.

Layer 2: $\;\dfrac{1}{4/3} = \tfrac{3}{4}$.

Layer 3: $\;1 - \tfrac{3}{4} = \tfrac{1}{4}$.

Outer: $\;\dfrac{1}{1/4} = 4$.

$4$
Problem 3
$\dfrac{1}{n(n+2)} = \dfrac{A}{n} + \dfrac{B}{n+2}$.
partial fractions gap 2 used in Part 3 #4

Combine: $\;A(n+2) + Bn = 1 \Rightarrow (A+B)n + 2A = 1$.

Match: $A+B = 0$, $2A = 1 \Rightarrow A = \tfrac{1}{2}$, $B = -\tfrac{1}{2}$.

$A = \tfrac{1}{2},\;\; B = -\tfrac{1}{2}$
Problem 4
$\dfrac{1}{(3n-1)(3n+2)} = \dfrac{A}{3n-1} + \dfrac{B}{3n+2}$.
partial fractions used in Part 3 #2

Combine: $\;A(3n+2) + B(3n-1) = 1 \Rightarrow (3A+3B)n + (2A - B) = 1$.

Match: $3A + 3B = 0 \Rightarrow A = -B$.  Then $2A - B = 2A - (-A) = 3A = 1 \Rightarrow A = \tfrac{1}{3}$, $B = -\tfrac{1}{3}$.

$A = \tfrac{1}{3},\;\; B = -\tfrac{1}{3}$
Problem 5 — Factor first
$\dfrac{1}{n^2 + 4n + 3} = \dfrac{A}{?} + \dfrac{B}{?}$.
factor + partial fractions used in Part 3 #3

Factor denominator: $\;n^2 + 4n + 3 = (n+1)(n+3)$.

Set $\;\dfrac{1}{(n+1)(n+3)} = \dfrac{A}{n+1} + \dfrac{B}{n+3}$. Combine: $A(n+3) + B(n+1) = 1$.

Match: $(A+B)n + (3A+B) = 1 \Rightarrow A+B=0$, $3A+B=1 \Rightarrow 2A=1 \Rightarrow A = \tfrac{1}{2}$, $B = -\tfrac{1}{2}$.

$A = \tfrac{1}{2},\;\; B = -\tfrac{1}{2}$   (over $(n+1)$ and $(n+3)$)
Part 2 — Geometric Series
Problem 1 — Finite
$\displaystyle\sum_{k=0}^{5} 6\left(\tfrac{2}{3}\right)^k$
finite formula 6 terms

$g_0 = 6$, $r = 2/3$, upper power $n+1 = 6$.

$\displaystyle S = 6 \cdot \frac{1 - (2/3)^6}{1 - 2/3} = 6 \cdot \frac{1 - 64/729}{1/3} = 6 \cdot 3 \cdot \tfrac{665}{729} = \tfrac{11970}{729} = \tfrac{1330}{81}$.

Decimal: $\approx 16.42$.

$S = \dfrac{1330}{81} \approx 16.42$
Problem 2 — Infinite
$\displaystyle\sum_{k=0}^{\infty} 6\left(\tfrac{2}{3}\right)^k$, plus the gap with #1.
infinite formula |r|<1 ✓

$|r| = 2/3 < 1$ ⇒ converges. $\displaystyle S_\infty = \tfrac{6}{1 - 2/3} = \tfrac{6}{1/3} = 18$.

Gap with #1: $\;18 - \tfrac{1330}{81} = \tfrac{1458}{81} - \tfrac{1330}{81} = \tfrac{128}{81} \approx 1.58$. That's the tail $\sum_{k=6}^{\infty} 6(2/3)^k$.

$S_\infty = 18$,   gap $= \dfrac{128}{81} \approx 1.58$
Problem 3 — Shifted start index
$\displaystyle\sum_{k=3}^{8}\left(\tfrac{1}{2}\right)^k$
finite index shift

Shift: let $j = k-3$, so $(1/2)^k = (1/2)^{j+3} = \tfrac{1}{8}(1/2)^j$. Sum becomes: $$\sum_{j=0}^{5} \tfrac{1}{8}(1/2)^j = \tfrac{1}{8} \cdot \frac{1 - (1/2)^6}{1 - 1/2} = \tfrac{1}{8} \cdot 2 \cdot \tfrac{63}{64} = \tfrac{63}{256}$$

Sanity (direct): $\;\tfrac{1}{8} + \tfrac{1}{16} + \tfrac{1}{32} + \tfrac{1}{64} + \tfrac{1}{128} + \tfrac{1}{256} = \tfrac{32+16+8+4+2+1}{256} = \tfrac{63}{256}$ ✓

$\dfrac{63}{256}$
Problem 4 — Negative ratio with $g_0 \ne 1$
$\displaystyle\sum_{k=0}^{\infty} 8\left(-\tfrac{3}{4}\right)^k$
infinite formula sign care

$g_0 = 8$, $r = -3/4$, $|r| = 3/4 < 1$ ✓.

$\displaystyle S_\infty = \frac{8}{1 - (-3/4)} = \frac{8}{1 + 3/4} = \frac{8}{7/4} = \frac{32}{7}$.

Sign of $r$ doesn't affect convergence — only $|r|$ does.

$S_\infty = \dfrac{32}{7}$
Problem 5 — 3-digit repeating decimal
Express $0.\overline{135} = 0.135135135\ldots$ as a fraction.
geometric series repeat block = 3 digits

Chunk by $3$ digits: $\;0.\overline{135} = 0.135 + 0.000135 + 0.000000135 + \cdots$

Geometric: $g_0 = \tfrac{135}{1000}$, $r = \tfrac{1}{1000}$.

$\displaystyle S_\infty = \frac{135/1000}{1 - 1/1000} = \frac{135/1000}{999/1000} = \frac{135}{999}$.

Reduce: $\;\gcd(135, 999) = 27$. So $\dfrac{135}{999} = \dfrac{5}{37}$.

Check: $5 \div 37 = 0.135135\ldots$ ✓

$0.\overline{135} = \dfrac{5}{37}$
Part 3 — Telescoping & Power Series IoC
Problem 1 — Telescoping (basic, gap-1)
$\displaystyle\sum_{n=1}^{\infty} \frac{1}{(2n-1)(2n+1)}$
telescoping uses Part 1 #4: A=½, B=−½

From Part 1 #4: $\;\dfrac{1}{(2n-1)(2n+1)} = \tfrac{1}{2}\!\left(\dfrac{1}{2n-1} - \dfrac{1}{2n+1}\right)$.

$\displaystyle S_N = \tfrac{1}{2}\Big[\Big(\tfrac{1}{1}-\tfrac{1}{3}\Big) + \Big(\tfrac{1}{3}-\tfrac{1}{5}\Big) + \cdots + \Big(\tfrac{1}{2N-1} - \tfrac{1}{2N+1}\Big)\Big] = \tfrac{1}{2}\!\left(1 - \tfrac{1}{2N+1}\right)$.

$N \to \infty$: $\;S_\infty = \tfrac{1}{2}(1 - 0) = \tfrac{1}{2}$.

$S_\infty = \dfrac{1}{2}$
Problem 2 — Telescoping (gap-1, new pattern)
$\displaystyle\sum_{n=1}^{\infty} \frac{1}{(3n-1)(3n+2)}$
telescoping uses Part 1 #4: A=⅓, B=−⅓

From Part 1 #4: $\;\dfrac{1}{(3n-1)(3n+2)} = \tfrac{1}{3}\!\left(\dfrac{1}{3n-1} - \dfrac{1}{3n+2}\right)$.

Plug in $n = 1, 2, 3, \ldots$: denominators are $\;(2,5),(5,8),(8,11),(11,14),\ldots$ Note $(3 \cdot 1 - 1) = 2$ and $(3n+2)$ for $n$ matches $(3(n+1) - 1)$ for the next term — perfect telescoping.

$\displaystyle S_N = \tfrac{1}{3}\Big[\Big(\tfrac{1}{2}-\tfrac{1}{5}\Big) + \Big(\tfrac{1}{5}-\tfrac{1}{8}\Big) + \cdots + \Big(\tfrac{1}{3N-1} - \tfrac{1}{3N+2}\Big)\Big] = \tfrac{1}{3}\!\left(\tfrac{1}{2} - \tfrac{1}{3N+2}\right)$.

$N \to \infty$: $\;S_\infty = \tfrac{1}{3} \cdot \tfrac{1}{2} = \tfrac{1}{6}$.

$S_\infty = \dfrac{1}{6}$
Problem 3 — Telescoping (gap-2, factor first)
$\displaystyle\sum_{n=1}^{\infty} \frac{1}{n^2+4n+3}$
factor first gap-2 telescoping uses Part 1 #5

Factor: $\;n^2 + 4n + 3 = (n+1)(n+3)$. From Part 1 #5: $\;\dfrac{1}{(n+1)(n+3)} = \tfrac{1}{2}\!\left(\dfrac{1}{n+1} - \dfrac{1}{n+3}\right)$.

Write out the first several terms (denominators jump by $2$): $$S_N = \tfrac{1}{2}\Big[\Big(\tfrac{1}{2}-\tfrac{1}{4}\Big)+\Big(\tfrac{1}{3}-\tfrac{1}{5}\Big)+\Big(\tfrac{1}{4}-\tfrac{1}{6}\Big)+\Big(\tfrac{1}{5}-\tfrac{1}{7}\Big)+\cdots+\Big(\tfrac{1}{N+1}-\tfrac{1}{N+3}\Big)\Big]$$

The $-\tfrac{1}{4}$ kills the later $+\tfrac{1}{4}$, the $-\tfrac{1}{5}$ kills the later $+\tfrac{1}{5}$, etc. — but with gap 2, two terms survive at the start ($\tfrac{1}{2}$ and $\tfrac{1}{3}$) and two die out at the end: $$S_N = \tfrac{1}{2}\!\left[\tfrac{1}{2} + \tfrac{1}{3} - \tfrac{1}{N+2} - \tfrac{1}{N+3}\right]$$

$N \to \infty$: $\;S_\infty = \tfrac{1}{2}\!\left(\tfrac{1}{2} + \tfrac{1}{3}\right) = \tfrac{1}{2} \cdot \tfrac{5}{6} = \tfrac{5}{12}$.

$S_\infty = \dfrac{5}{12}$
Problem 4 — Telescoping (gap-2)
$\displaystyle\sum_{n=1}^{\infty} \frac{1}{n(n+2)}$
gap-2 telescoping uses Part 1 #3

From Part 1 #3: $\;\dfrac{1}{n(n+2)} = \tfrac{1}{2}\!\left(\dfrac{1}{n} - \dfrac{1}{n+2}\right)$.

Write out (gap 2 again): $$S_N = \tfrac{1}{2}\Big[\Big(1-\tfrac{1}{3}\Big)+\Big(\tfrac{1}{2}-\tfrac{1}{4}\Big)+\Big(\tfrac{1}{3}-\tfrac{1}{5}\Big)+\Big(\tfrac{1}{4}-\tfrac{1}{6}\Big)+\cdots+\Big(\tfrac{1}{N}-\tfrac{1}{N+2}\Big)\Big]$$

Survivors: first two ($1$ and $\tfrac{1}{2}$), and the last two negative terms vanish. $$S_N = \tfrac{1}{2}\!\left[1 + \tfrac{1}{2} - \tfrac{1}{N+1} - \tfrac{1}{N+2}\right]$$

$N \to \infty$: $\;S_\infty = \tfrac{1}{2}\!\left(1 + \tfrac{1}{2}\right) = \tfrac{1}{2} \cdot \tfrac{3}{2} = \tfrac{3}{4}$.

$S_\infty = \dfrac{3}{4}$
Problem 5 — FINAL CHALLENGE: Power Series IoC
Find IoC and $P(x)$ for $\displaystyle\sum_{k=0}^{\infty}\left(\tfrac{x}{2}\right)^k$.
power series IoC complex fraction (Part 1)

Step 1 — Identify $g_0$ and $r(x)$. At $k=0$: $g_0 = (x/2)^0 = 1$. Ratio $r(x) = \tfrac{x}{2}$.

Step 2 — Convergence. $$\left|\tfrac{x}{2}\right| < 1 \;\Longrightarrow\; \tfrac{|x|}{2} < 1 \;\Longrightarrow\; |x| < 2 \;\Longrightarrow\; -2 < x < 2$$ IoC $= (-2, 2)$.

Step 3 — Sum function (clean up the complex fraction). $$P(x) = \frac{g_0}{1 - r(x)} = \frac{1}{1 - x/2}$$ Multiply top and bottom by $2$ (Part-1 trick): $$P(x) = \frac{1 \cdot 2}{(1 - x/2)\cdot 2} = \frac{2}{2 - x}$$

Sanity checks:

• $x = 0$: series $= 1$, formula $= \tfrac{2}{2} = 1$ ✓.

• $x = 1$: series $= 1 + \tfrac{1}{2} + \tfrac{1}{4} + \cdots = 2$, formula $= \tfrac{2}{2-1} = 2$ ✓.

IoC $= (-2, 2)$,   $P(x) = \dfrac{2}{2-x}$
