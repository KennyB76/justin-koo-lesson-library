Item 1 · Power Series as a Family of Geometric Series
Problem 1.1
$\sum_{k=0}^{\infty} (x/2)^k$ at $x = 1$. Converges? Sum?
Plug in |r|<1
At $x=1$: ratio $r = 1/2$. Since $|r| = 1/2 < 1$, the geometric series converges.
$g_0 = 1$, so $\displaystyle \text{Sum} = \frac{g_0}{1-r} = \frac{1}{1 - 1/2} = 2.$
Converges; sum = 2
Problem 1.2
$\sum_{k=0}^{\infty} (3x)^k$ — give one converging $x$, one diverging $x$.
|r|<1
$r = 3x$. Convergence requires $|3x| < 1 \;\Longleftrightarrow\; |x| < \tfrac{1}{3}$.
Converges: $x = \tfrac{1}{4}$ → $|r| = 3/4 < 1$. ✓
Diverges: $x = 1$ → $|r| = 3 > 1$. ✗ (any $|x| \ge 1/3$ works).
e.g. $x=\tfrac{1}{4}$ converges, $x=1$ diverges
Problem 1.3
$\sum_{k=0}^{\infty} \left(\frac{x-2}{4}\right)^k$ — identify $r$ and $g_0$.
Identify ratio
Index starts at $k=0$, so the first term is $\left(\frac{x-2}{4}\right)^0 = 1 \;\Rightarrow\; g_0 = 1$.
Ratio $r = \dfrac{x-2}{4}$.
$g_0 = 1, \;\; r = \dfrac{x-2}{4}$
Problem 1.4
$\sum_{k=1}^{\infty} \left(\frac{x+1}{3}\right)^k$ — identify $g_0$ and $r$ (note $k$ starts at 1!).
Index starts ≠ 0
Since $k$ starts at 1, the first term is $\left(\frac{x+1}{3}\right)^1 = \frac{x+1}{3}$. So $g_0 = \dfrac{x+1}{3}$.
Ratio between consecutive terms is still $r = \dfrac{x+1}{3}$.
Note: Sum (when convergent) is $\dfrac{g_0}{1-r}$, not $\dfrac{1}{1-r}$.
$g_0 = \dfrac{x+1}{3}, \;\; r = \dfrac{x+1}{3}$
Item 2 · Find IoC and Sum Function $P(x)$
Problem 2.1
$\sum_{k=0}^{\infty}(x/5)^k$ → IoC and $P(x)$.
|r|<1 P = g₀/(1−r)
$g_0 = 1$, $r = x/5$. Need $|x/5| < 1 \;\Rightarrow\; |x| < 5$.
$P(x) = \dfrac{1}{1 - x/5} = \dfrac{5}{5 - x}.$
IoC $= (-5, 5),\; P(x) = \dfrac{5}{5-x}$
Problem 2.2
$\sum_{k=0}^{\infty}\left(\frac{x-1}{3}\right)^k$ → IoC and $P(x)$.
Shifted center
$g_0 = 1$, $r = \dfrac{x-1}{3}$. Need $\left|\dfrac{x-1}{3}\right| < 1 \;\Rightarrow\; |x-1| < 3 \;\Rightarrow\; -3 < x-1 < 3 \;\Rightarrow\; -2 < x < 4$.
$P(x) = \dfrac{1}{1 - (x-1)/3} = \dfrac{3}{3 - (x-1)} = \dfrac{3}{4 - x}.$
IoC $= (-2, 4),\; P(x) = \dfrac{3}{4-x}$
Problem 2.3
$\sum_{k=0}^{\infty} (3/x)^k$ → IoC and $P(x)$.
Variable in denom
$g_0 = 1$, $r = 3/x$. Need $\left|\dfrac{3}{x}\right| < 1 \;\Rightarrow\; \dfrac{3}{|x|} < 1 \;\Rightarrow\; |x| > 3$.
So IoC = $(-\infty, -3) \cup (3, \infty)$ — two rays.
$P(x) = \dfrac{1}{1 - 3/x} = \dfrac{x}{x - 3}.$
IoC $= (-\infty, -3) \cup (3, \infty),\; P(x) = \dfrac{x}{x-3}$
Problem 2.4
$\sum_{k=1}^{\infty}(x/4)^k$ → IoC and $P(x)$. (Starts at $k=1$.)
k=1 start g₀ ≠ 1
First term ($k=1$): $g_0 = \dfrac{x}{4}$. Ratio $r = \dfrac{x}{4}$. Convergence: $|x/4| < 1 \;\Rightarrow\; |x| < 4$.
$P(x) = \dfrac{g_0}{1-r} = \dfrac{x/4}{1 - x/4} = \dfrac{x/4}{(4-x)/4} = \dfrac{x}{4 - x}.$
IoC $= (-4, 4),\; P(x) = \dfrac{x}{4-x}$
Item 3 · Three Σ Rules
Problem 3.1 — Pull out
Rewrite $\sum_{n=1}^{8} 5n$ with the constant out.
Rule I
$\displaystyle \sum_{n=1}^{8} 5n = 5 \sum_{n=1}^{8} n.$
$5 \displaystyle\sum_{n=1}^{8} n$
Problem 3.2 — Split
Rewrite $\sum_{n=1}^{12}(n^2 - 3n + 4)$ as 3 separate sums.
Rule II Rule I
First split (Rule II), then pull constants (Rule I):
$\displaystyle \sum_{n=1}^{12} (n^2 - 3n + 4) = \sum_{n=1}^{12} n^2 \; - \; 3\sum_{n=1}^{12} n \; + \; \sum_{n=1}^{12} 4.$
Bonus simplification: $\sum_{n=1}^{12} 4 = 12 \cdot 4 = 48$.
$\displaystyle\sum_{n=1}^{12} n^2 - 3\sum_{n=1}^{12} n + \sum_{n=1}^{12} 4$
Problem 3.3 — Index shift
Rewrite $\sum_{n=3}^{20} n^2$ to start at $n=0$.
Rule III
Substitute $n \to n+3$ inside the summand. Drop the start by 3 (so 3 → 0, 20 → 17):
$\displaystyle \sum_{n=3}^{20} n^2 = \sum_{n=0}^{17} (n+3)^2.$
Sanity check: new $n=0$ → $(0+3)^2 = 9 = 3^2$ ✓; new $n=17$ → $(17+3)^2 = 400 = 20^2$ ✓.
$\displaystyle\sum_{n=0}^{17} (n+3)^2$
Problem 3.4 — Combined
Rewrite $\sum_{n=5}^{14} 4(n-3)^2$ — pull out 4, then shift to $n=0$.
Rule I + III
Step 1 (pull out 4): $\displaystyle 4 \sum_{n=5}^{14} (n-3)^2.$
Step 2 (shift $n \to n+5$, drop bounds by 5): $\displaystyle 4 \sum_{n=0}^{9} \big((n+5)-3\big)^2 = 4 \sum_{n=0}^{9} (n+2)^2.$
Sanity check: new $n=0$ → $(0+2)^2 = 4 = (5-3)^2$ ✓; new $n=9$ → $(9+2)^2 = 121 = (14-3)^2$ ✓.
$4\displaystyle\sum_{n=0}^{9}(n+2)^2$
Item 4 · Bernoulli Telescoping
Problem 4.1 — Set up partial fractions
Find $A, B$: $\dfrac{1}{(n+1)(n+2)} = \dfrac{A}{n+1} + \dfrac{B}{n+2}$.
Partial fractions
Multiply through by $(n+1)(n+2)$: $\;1 = A(n+2) + B(n+1)$.
Plug $n = -1$: $\;1 = A(1) + B(0) \;\Rightarrow\; A = 1$.
Plug $n = -2$: $\;1 = A(0) + B(-1) \;\Rightarrow\; B = -1$.
$A = 1, \;\; B = -1$
Problem 4.2 — Telescope to ∞
Evaluate $\sum_{n=1}^{\infty} \dfrac{1}{(n+1)(n+2)}$.
Telescoping Limit
From 4.1: each term $= \dfrac{1}{n+1} - \dfrac{1}{n+2}$.
Partial sum $S_N = \left(\dfrac{1}{2} - \dfrac{1}{3}\right) + \left(\dfrac{1}{3} - \dfrac{1}{4}\right) + \cdots + \left(\dfrac{1}{N+1} - \dfrac{1}{N+2}\right) = \dfrac{1}{2} - \dfrac{1}{N+2}.$
As $N \to \infty$: $\dfrac{1}{N+2} \to 0$, so $S_\infty = \dfrac{1}{2}$.
$\dfrac{1}{2}$
Problem 4.3 — Bernoulli style
Evaluate $\sum_{n=1}^{\infty} \dfrac{1}{(4n-3)(4n+1)}$.
Partial fractions Telescoping
Set $\dfrac{1}{(4n-3)(4n+1)} = \dfrac{A}{4n-3} + \dfrac{B}{4n+1}$. Multiply: $\;1 = A(4n+1) + B(4n-3)$.
$n$ coefficient: $4A + 4B = 0 \;\Rightarrow\; A = -B$. Constant: $A - 3B = 1 \;\Rightarrow\; -B - 3B = 1 \;\Rightarrow\; B = -\tfrac{1}{4},\; A = \tfrac{1}{4}.$
Each term $= \dfrac{1}{4}\left(\dfrac{1}{4n-3} - \dfrac{1}{4n+1}\right)$. Partial sum:
$S_N = \dfrac{1}{4}\!\left[\!\left(\dfrac{1}{1} - \dfrac{1}{5}\right) + \left(\dfrac{1}{5} - \dfrac{1}{9}\right) + \cdots + \left(\dfrac{1}{4N-3} - \dfrac{1}{4N+1}\right)\!\right] = \dfrac{1}{4}\!\left(1 - \dfrac{1}{4N+1}\right).$
As $N \to \infty$: $S_\infty = \dfrac{1}{4}(1 - 0) = \dfrac{1}{4}$.
$\dfrac{1}{4}$
Problem 4.4 — Finite telescoping
Compute $\sum_{n=1}^{4} \dfrac{1}{n(n+1)}$ using $\dfrac{1}{n(n+1)} = \dfrac{1}{n} - \dfrac{1}{n+1}$.
Finite sum
Write out all 4 terms:
$n=1$: $\;1 - \tfrac{1}{2}$
$n=2$: $\;\tfrac{1}{2} - \tfrac{1}{3}$
$n=3$: $\;\tfrac{1}{3} - \tfrac{1}{4}$
$n=4$: $\;\tfrac{1}{4} - \tfrac{1}{5}$
Summing: $\;1 + (-\tfrac{1}{2} + \tfrac{1}{2}) + (-\tfrac{1}{3} + \tfrac{1}{3}) + (-\tfrac{1}{4} + \tfrac{1}{4}) - \tfrac{1}{5} = 1 - \tfrac{1}{5} = \tfrac{4}{5}.$
$\dfrac{4}{5}$
