Sequence of Partial Sums & Geometric Series Convergence

Student: Justin Koo Date: 2026-05-03
Standard: CCSS.MATH.CONTENT.HSF-BF.A.2 / HSF-IF.A.3 — Build and analyze sequences/series, including geometric series and their convergence behavior.
1. Concept Insights — Why Partial Sums?
The core problem

You can't literally add infinitely many numbers — that's not an arithmetic operation. So we cheat: we look at the finite sums $S_1, S_2, S_3, \ldots$ and watch what they do.

This new list — $S_n$ — is called the sequence of partial sums:

$\displaystyle S_n = \sum_{k=1}^{n} T_k = T_1 + T_2 + \cdots + T_n$

Convergence definition: the series $\sum T_k$ converges $\iff$ $\displaystyle\lim_{n\to\infty} S_n$ exists (and is finite). Otherwise it diverges.

Geometric series — the one you can fully solve

For a geometric series $g_0 + g_0 r + g_0 r^2 + \cdots$ with first term $g_0$ and ratio $r$:

$\displaystyle S_n = g_0 \cdot \frac{1 - r^n}{1 - r}$

The whole game is what $r^n$ does as $n \to \infty$:

  • $|r| < 1 \;\Rightarrow\; r^n \to 0 \;\Rightarrow\; S_\infty = \dfrac{g_0}{1 - r}$   (converges)
  • $|r| \ge 1 \;\Rightarrow\; r^n$ does NOT die $\;\Rightarrow\;$ diverges
Visual intuition (Kenny's footstep model)
  • $|r| < 1$: each step is shrinking. You walk forever but cover a finite distance — like Zeno's paradox.
  • $|r| > 1$: each step is bigger than the last. You explode off to infinity.
  • $r = 1$: every step is the same size — you never stop adding the same number.
  • $r = -1$: you flip-flop forever — $S_n$ oscillates, no limit.
2. Worked Example — Simple
Example 1 — A clean convergent geometric series
Evaluate $\displaystyle \sum_{k=1}^{\infty} \frac{3}{10^k} = \frac{3}{10} + \frac{3}{100} + \frac{3}{1000} + \cdots$

Step 1 — Identify the geometric structure. First term is $g_0 = \tfrac{3}{10}$, ratio $r = \tfrac{1}{10}$.

Step 2 — Build the partial sum.

$\displaystyle S_n = \frac{3}{10} \cdot \frac{1 - (1/10)^n}{1 - 1/10} = \frac{1}{3}\left(1 - \left(\tfrac{1}{10}\right)^n\right)$

Step 3 — Take the limit. Since $|r| = 1/10 < 1$, $\;(1/10)^n \to 0$.

$\displaystyle S_\infty = \frac{1}{3}(1 - 0) = \frac{1}{3}$

Answer: $S_\infty = \dfrac{1}{3}$.

Sanity check: $0.3 + 0.03 + 0.003 + \cdots = 0.333\ldots = 1/3$. ✓

3. Worked Example — Harder (Watch for the Trap)
Example 2 — When the series blows up
Determine whether the series converges, and if so find its sum: $$2 + 3 + \frac{9}{2} + \frac{27}{4} + \frac{81}{8} + \cdots$$

Step 1 — Hunt for the ratio. Divide consecutive terms: $$\frac{3}{2}, \;\; \frac{9/2}{3} = \frac{3}{2}, \;\; \frac{27/4}{9/2} = \frac{3}{2}, \;\ldots$$ So $r = \tfrac{3}{2}$ and $g_0 = 2$. We can rewrite as $\;2 + 2(\tfrac{3}{2}) + 2(\tfrac{3}{2})^2 + \cdots$.

Step 2 — Apply the convergence test. Is $|r| < 1$?   $|3/2| = 1.5 \;\not<\; 1$. So $r^n \to \infty$, not $0$.

Step 3 — Confirm with the partial sum. $$S_n = 2 \cdot \frac{1 - (3/2)^n}{1 - 3/2} = 2 \cdot \frac{1 - (3/2)^n}{-1/2} = 4\big((3/2)^n - 1\big)$$ As $n \to \infty$, $(3/2)^n \to \infty$, so $S_n \to \infty$.

Answer: The series diverges. No finite sum.

Teaching question: "Why doesn't the formula $g_0/(1-r)$ work here?" → Because the formula only exists when $|r|<1$. Plugging $r=3/2$ gives $\tfrac{2}{1-3/2} = -4$, which is meaningless — the series doesn't equal $-4$, it equals $+\infty$. The formula is a limit shortcut, not algebra.

4. Practice — 5 Problems with Answers Below
Try each problem first. The answer + reasoning is in the green block immediately below. Cover it with your hand, work it out, then check.
1. Evaluate $\displaystyle \sum_{k=1}^{\infty} 5\left(\tfrac{1}{4}\right)^k$.
ANSWER

First term (when $k=1$): $g_0 = 5 \cdot \tfrac{1}{4} = \tfrac{5}{4}$. Ratio $r = \tfrac{1}{4}$, $|r|<1$ ✓.

$\displaystyle S_\infty = \frac{g_0}{1-r} = \frac{5/4}{1 - 1/4} = \frac{5/4}{3/4} = \frac{5}{3}$.

$S_\infty = \dfrac{5}{3}$

2. Evaluate $\displaystyle \sum_{k=1}^{\infty} 6\left(\tfrac{1}{3}\right)^{k-1}$. (Watch the exponent — this starts at $k=1$ but uses $k-1$.)
ANSWER

When $k=1$, the exponent is $0$, so the first term is $6 \cdot 1 = 6$. So $g_0 = 6$, $r = \tfrac{1}{3}$.

$\displaystyle S_\infty = \frac{6}{1 - 1/3} = \frac{6}{2/3} = 9$.

$S_\infty = 9$

Trap warning: if you mistakenly took $g_0 = 6 \cdot 1/3 = 2$ (treating it like $r^k$), you'd get $3$ — wrong. Always plug in the starting index.

3. Find the exact sum of $\;0.7 + 0.49 + 0.343 + 0.2401 + \cdots$
ANSWER

Ratio: $\tfrac{0.49}{0.7} = 0.7$. Confirm: $\tfrac{0.343}{0.49} = 0.7$ ✓. So $g_0 = 0.7$, $r = 0.7$, $|r|<1$ ✓.

$\displaystyle S_\infty = \frac{0.7}{1 - 0.7} = \frac{0.7}{0.3} = \frac{7}{3}$.

$S_\infty = \dfrac{7}{3} \approx 2.333\ldots$

4. Evaluate $\displaystyle \sum_{k=0}^{\infty} \left(-\tfrac{2}{5}\right)^k$.  (Negative ratio — series oscillates.)
ANSWER

Starts at $k=0$: first term $= (-2/5)^0 = 1$. So $g_0 = 1$, $r = -\tfrac{2}{5}$. Check $|r| = 2/5 < 1$ ✓.

$\displaystyle S_\infty = \frac{1}{1 - (-2/5)} = \frac{1}{1 + 2/5} = \frac{1}{7/5} = \frac{5}{7}$.

$S_\infty = \dfrac{5}{7}$

Notice: negative $r$ doesn't break convergence — only $|r| \ge 1$ does. The terms alternate sign but their magnitudes still shrink.

5. Convert the repeating decimal $0.272727\ldots$ into an exact fraction using a geometric series.
ANSWER

Write it as a sum: $\;0.27 + 0.0027 + 0.000027 + \cdots$

That's geometric with $g_0 = 0.27 = \tfrac{27}{100}$ and $r = \tfrac{1}{100}$ (each chunk is $1/100$ of the last).

$\displaystyle S_\infty = \frac{27/100}{1 - 1/100} = \frac{27/100}{99/100} = \frac{27}{99} = \frac{3}{11}$.

$0.\overline{27} = \dfrac{3}{11}$

This trick works for any repeating decimal — that's why every repeating decimal is rational.
