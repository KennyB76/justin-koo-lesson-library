Power Series, Interval of Convergence, Σ Rules & Bernoulli (Telescoping) Sums

Student: Justin Koo Date: 2026-05-03
Standard: CCSS.MATH.CONTENT.HSF-BF.A.2 / HSA-SSE.B.4 — Recognize and rewrite series; identify intervals of convergence for parametric (power) series; manipulate finite sums.
1. Concept Insights — Three Big Ideas
Idea 1: A Power Series is a "family" of geometric series, indexed by $x$

If we write $\displaystyle \sum_{k=0}^{\infty} \left(\frac{2x}{3}\right)^k$, we get a different infinite geometric series for every $x$ we plug in:

  • $x = 1$: ratio $= \tfrac{2}{3}$, $|r| < 1$ → converges.
  • $x = 2$: ratio $= \tfrac{4}{3}$, $|r| > 1$ → diverges.

The set of $x$-values for which the series converges is the Interval of Convergence (IoC), also called the Domain of Convergence. Inside that interval, the series defines a function $P(x)$.

Idea 2: How to find the IoC — apply $|r|<1$ where $r$ depends on $x$

Identify the ratio expression in $x$, set $|\,r(x)\,|<1$, solve for $x$:

$\displaystyle \sum (r(x))^k \;\text{ converges } \iff |r(x)| < 1$

Then the closed-form sum on the IoC is the geometric formula: $\;P(x) = \dfrac{g_0(x)}{1 - r(x)}$.

Idea 3: Three rules for manipulating Σ (algebra of finite sums)
RuleStatementUse it when…
(I) Constant pull-out $\displaystyle \sum_{a=1}^{b} c \cdot f(a) = c \cdot \sum_{a=1}^{b} f(a)$ A constant multiplies every term — pull it out front.
(II) Sum split $\displaystyle \sum (f(a) + g(a)) = \sum f(a) + \sum g(a)$ The summand splits into two pieces you know how to sum separately.
(III) Index shift $\displaystyle \sum_{n=7}^{20} (n^2+n) = \sum_{n=0}^{13} \big[(n+7)^2 + (n+7)\big]$ You want the index to start at $0$ or $1$ — substitute $n \to n+7$ inside, drop the start by $7$.
Idea 4: Bernoulli's Telescoping Trick

For a sum like $\displaystyle\sum \frac{1}{(3n-2)(3n+1)}$, use partial fractions to split each term into a difference $\frac{A}{\cdot} - \frac{B}{\cdot}$. When you write out all the terms, almost everything cancels in pairs — only the very first and very last survive. That's a "telescoping" sum.

Why "Bernoulli madness"? Because Jakob Bernoulli used this trick to crack sums that everyone thought were impossible. The leverage is huge — a hard infinite sum collapses into one or two surviving terms.

2. Worked Example — Simple (Find the IoC)
Example 1 — A power series in disguise
For the series $\displaystyle 1 - \frac{2}{x} + \frac{4}{x^2} - \frac{8}{x^3} + \cdots$, find the Interval of Convergence and the sum function $P(x)$.

Step 1 — Spot the geometric pattern. Each term is the previous times $-\tfrac{2}{x}$. Rewrite: $$\sum_{k=0}^{\infty} \left(-\tfrac{2}{x}\right)^k$$ So $g_0 = 1$ and $r = -\tfrac{2}{x}$.

Step 2 — Convergence condition. Need $|r| < 1$: $$\left|-\tfrac{2}{x}\right| < 1 \;\Longrightarrow\; \frac{2}{|x|} < 1 \;\Longrightarrow\; |x| > 2$$ So $x > 2$ or $x < -2$, i.e. $\;x \in (-\infty, -2) \cup (2, \infty)$.

Step 3 — Find the sum function on the IoC. $$P(x) = \frac{g_0}{1 - r} = \frac{1}{1 - (-2/x)} = \frac{1}{1 + 2/x} = \frac{x}{x+2}$$

IoC: $(-\infty, -2) \cup (2, \infty)$.   Sum: $P(x) = \dfrac{x}{x+2}$.

3. Worked Example — Harder (Bernoulli Telescoping)
Example 2 — The "freakadelic" power series + a telescoping sum
(a) Find the IoC and sum function for $\displaystyle \sum_{k=1}^{\infty} \left(\tfrac{2x}{3}\right)^k$.   (b) Evaluate the telescoping sum $\displaystyle \sum_{n=1}^{\infty} \frac{1}{(3n-2)(3n+1)}$.

(a) Power series. The series starts at $k=1$, so the first term is $g_0 = \tfrac{2x}{3}$ (not $1$). Ratio $r = \tfrac{2x}{3}$.

Need $|r|<1$: $\left|\tfrac{2x}{3}\right| < 1 \Longrightarrow |2x| < 3 \Longrightarrow |x| < \tfrac{3}{2}$. IoC $= \left(-\tfrac{3}{2}, \tfrac{3}{2}\right)$.

Sum: $\displaystyle P(x) = \frac{g_0}{1-r} = \frac{2x/3}{1 - 2x/3} = \frac{2x/3}{(3-2x)/3} = \frac{2x}{3 - 2x}$.

(a) IoC $= \big(-\tfrac{3}{2}, \tfrac{3}{2}\big)$,   $P(x) = \dfrac{2x}{3-2x}$.

(b) Telescoping. Set up partial fractions: $$\frac{1}{(3n-2)(3n+1)} = \frac{A}{3n-2} + \frac{B}{3n+1}$$ Multiply both sides by $(3n-2)(3n+1)$:   $1 = A(3n+1) + B(3n-2)$.

This must hold for every $n$, so match coefficients of $n$ and the constant: $$\begin{cases} 3A + 3B = 0 & (n\text{ coeff}) \\ A - 2B = 1 & (\text{const}) \end{cases} \;\Longrightarrow\; A = -B,\; -3B = 1 \cdot \text{(some algebra)} \;\Longrightarrow\; A = \tfrac{1}{3},\; B = -\tfrac{1}{3}$$

So each term equals $\;\frac{1}{3}\!\left(\frac{1}{3n-2} - \frac{1}{3n+1}\right)$. Summing $n=1$ to $N$: $$S_N = \frac{1}{3}\Big[\Big(\tfrac{1}{1} - \tfrac{1}{4}\Big) + \Big(\tfrac{1}{4} - \tfrac{1}{7}\Big) + \Big(\tfrac{1}{7} - \tfrac{1}{10}\Big) + \cdots + \Big(\tfrac{1}{3N-2} - \tfrac{1}{3N+1}\Big)\Big]$$

Everything in the middle telescopes — only the first and last survive: $$S_N = \frac{1}{3}\left(1 - \frac{1}{3N+1}\right)$$

As $N \to \infty$, $\tfrac{1}{3N+1} \to 0$, so: $$S_\infty = \frac{1}{3}(1 - 0) = \frac{1}{3}$$

(b) Sum $= \dfrac{1}{3}$.

4. Practice — 5 Problems with Answers Below
Cover the green block, work each one out, then check.
1. Find the Interval of Convergence and the sum function for $\displaystyle \sum_{k=0}^{\infty}\left(\tfrac{x}{4}\right)^k$.
ANSWER

$g_0 = 1$ (since $k=0$ gives $1$), $r = \tfrac{x}{4}$. Need $\left|\tfrac{x}{4}\right| < 1 \Rightarrow |x|<4$.

$\displaystyle P(x) = \frac{1}{1 - x/4} = \frac{4}{4-x}$.

IoC $= (-4, 4)$,   $P(x) = \dfrac{4}{4-x}$

2. Find the IoC and sum function for $\displaystyle \sum_{k=0}^{\infty}\left(\tfrac{-x}{2}\right)^k$. (Negative inside the parentheses — be careful with the absolute value.)
ANSWER

$g_0 = 1$, $r = -\tfrac{x}{2}$. Convergence: $\left|-\tfrac{x}{2}\right| = \tfrac{|x|}{2} < 1 \Rightarrow |x|<2$.

$\displaystyle P(x) = \frac{1}{1 - (-x/2)} = \frac{1}{1 + x/2} = \frac{2}{2+x}$.

IoC $= (-2, 2)$,   $P(x) = \dfrac{2}{2+x}$

3. Rewrite $\displaystyle \sum_{k=5}^{15} k^2$ so the index starts at $k = 0$. Do not evaluate.
ANSWER

Substitute $k \to k+5$ inside the summand and shift the bounds down by $5$:

$\displaystyle \sum_{k=5}^{15} k^2 = \sum_{k=0}^{10} (k+5)^2$

$\displaystyle \sum_{k=0}^{10}(k+5)^2$

Sanity check: when the new $k=0$, summand $= 25 = 5^2$ ✓. When new $k=10$, summand $= 225 = 15^2$ ✓.

4. Evaluate the telescoping sum $\displaystyle \sum_{n=1}^{\infty} \frac{1}{n(n+1)}$.
ANSWER

Partial fractions: $\dfrac{1}{n(n+1)} = \dfrac{1}{n} - \dfrac{1}{n+1}$ (verify by combining: $\tfrac{(n+1) - n}{n(n+1)} = \tfrac{1}{n(n+1)}$ ✓).

Partial sum: $$S_N = \Big(1 - \tfrac{1}{2}\Big) + \Big(\tfrac{1}{2} - \tfrac{1}{3}\Big) + \cdots + \Big(\tfrac{1}{N} - \tfrac{1}{N+1}\Big) = 1 - \tfrac{1}{N+1}$$

As $N \to \infty$: $\;S_\infty = 1 - 0 = 1$.

Sum $= 1$

5. Evaluate $\displaystyle \sum_{n=1}^{\infty} \frac{1}{(2n-1)(2n+1)}$   (Bernoulli style — find $A$, $B$ first.)
ANSWER

$\dfrac{1}{(2n-1)(2n+1)} = \dfrac{A}{2n-1} + \dfrac{B}{2n+1}$. Combining: $1 = A(2n+1) + B(2n-1)$.

Match: $\;2A + 2B = 0 \Rightarrow A = -B$.   Constant: $\;A - B = 1 \Rightarrow 2A = 1 \Rightarrow A = \tfrac{1}{2},\; B = -\tfrac{1}{2}$.

So each term $= \tfrac{1}{2}\!\left(\tfrac{1}{2n-1} - \tfrac{1}{2n+1}\right)$. Telescoping partial sum: $$S_N = \tfrac{1}{2}\Big[\Big(1 - \tfrac{1}{3}\Big) + \Big(\tfrac{1}{3} - \tfrac{1}{5}\Big) + \cdots + \Big(\tfrac{1}{2N-1} - \tfrac{1}{2N+1}\Big)\Big] = \tfrac{1}{2}\Big(1 - \tfrac{1}{2N+1}\Big)$$

$N \to \infty$: $\;S_\infty = \tfrac{1}{2}(1 - 0) = \tfrac{1}{2}$.

Sum $= \dfrac{1}{2}$

5. Workout — Finite vs Infinite Geometric Sums
READ FIRST. A finite sum $\displaystyle\sum_{k=0}^{n} r^k = \dfrac{1 - r^{\,n+1}}{1 - r}\;$ always exists (no condition on $r$, as long as $r \ne 1$). An infinite sum $\displaystyle\sum_{k=0}^{\infty} r^k = \dfrac{1}{1-r}\;$ exists only when $|r| < 1$ — otherwise the series diverges (write "DIVERGES").
